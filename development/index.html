The Simplest AngularJS Google Maps V3 API Directive

There is already one for this. However, I found myself doing totally different approach for this purpose than the existing one, such as;

  1. Everything in tag and attributes. Therefore, basic users does not even have to know what Javascript is. Tag and attribute does it all.
  2. Expose the original Google Maps V3 api to the user through this directive. No hiding nor wraping. By doing so, users don't need to learn this module again to use Google Map. You only need to know Google Maps V3 API.
  • map google map
  • marker marker on a map
  • shape circle, rectangle, or image on a map

To use this in your app, simply include javascript 

            ^lt;script src="http://maps.google.com/maps/api/js?sensor=false"></script>
            <script src="ng-map.min.js"></script>
